# ADR-003 — PostgreSQL

**Status:** Accepted · 2026-06-06

## Context

Pre-selected. Relational data with strong integrity needs (stock journal, price history, conversions).

## Decision

PostgreSQL as the system of record, accessed from the domain via a persistence layer (repositories).

## Consequences

Transactions enforce aggregate consistency boundaries. Rich types (numeric, timestamptz, jsonb) cover quantities, expiry, and semi-structured AI output. No domain logic lives in the database.
