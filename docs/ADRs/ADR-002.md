# ADR-002 — .NET backend

**Status:** Accepted · 2026-06-06

## Context

Pre-selected; the developer is backend-first and most productive in .NET.

## Decision

.NET hosts the domain model and application services.

## Consequences

A mature DI, persistence (EF Core), and testing story for the domain. Critically, having a capable server tier means we do **not** need a second server tier in JavaScript — this directly shapes ADR-004 and ADR-005.
